# ----------> Commands to run Silver Tier <----------



## For Staring Gmail Watcher Use this command 
python watchers/gmail_watcher.py AI_Employee_Vault --interval 60

## For Starting LinkedIn Watcher Use this command
python watchers/linkedin_poster.py AI_Employee_Vault ^ --content "Paste Your Post Content Here"

###  3. Open Obsidian → Pending_Approval/

###  4. Review the draft (it will have proper content now)

###  5. Move to Approved/

###  6. Run: python orchestrator.py AI_Employee_Vault --process-approvals

###  7. Check your sent emails - the full draft should be there! 📧
